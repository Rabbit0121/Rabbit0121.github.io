CF1948C Path Arrow

题目链接：CF1948C Path Arrow

大致题意

给出一个由 ‘ < ‘ 和 ‘ > ‘ 组成的 $2×n$ 的地图，其中 ‘ < ‘ 和 ‘ > ‘ 代表前进方向，问机器人从 $(1,1)$ 出发能否到达 $(2,n)$ 处。

每秒钟，以下两个动作依次发生：

  • 首先，机器人向左、右、下或上移动 (它不能尝试走出网格，也不能跳过移动)；
  • 然后沿着放置在当前单元格中的箭头移动（它在移动后停留在的单元格）

输入的额外约束条件：

  • $n$ 是偶数；
  • 没有箭头指向网格外部；

解题思路

看到地图、到达 、最短步数等字眼应该想到BFS算法，将从当前位置出发，下一步可能到达的所有位置入队，如果在队空之前能到达目标点，输出 $YES$，否则输出 $NO$
